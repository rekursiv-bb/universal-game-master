# Universal Game Master

## Background

This is the second version of the Universal Game Master, recreated from scratch using OpenAI's ChatGPT 5.2 Thinking model. It uses some of the principles behind the first handmade version, but attempts to resolve increasing issues cropping up with further AI guidance.

## Base Instructions

The instructions contained with `instructions.md` are intended to be directly input, either into the chat directly, or into the instructions field of the custom agent.
